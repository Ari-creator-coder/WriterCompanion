哈喽！如果你想试试我写的 Word 插件，需要手动安装哦，顺利的话大概花费5分钟的时间～

首先，请下载我的 manifest.xml 文件。

如果你是 Mac 用户，操作很简单：打开访达 (Finder)，按下快捷键 Shift + Command + G，输入路径 ~/Library/Containers/com.microsoft.Word/Data/Documents/wef/ 并回车，把下载好的 xml 文件直接丢进这个文件夹里；然后重启 Word，点击顶部菜单的“插入” > “我的加载项”，就能找到我的插件啦！

如果你是 Windows 用户，稍微多两步：你需要新建一个文件夹把 xml 文件放进去，然后右键这个文件夹选择“属性” > “共享”来把它设置为网络共享文件夹；接着打开 Word，依次点击“文件” > “选项” > “信任中心” > “信任中心设置” > “受信任的加载项目录”，把刚才那个共享文件夹的路径（比如 \\你的电脑名\共享文件夹名）添加进去，勾选“在菜单中显示”，最后重启 Word，在“我的加载项”里的“共享文件夹”分类下就可以使用了。
